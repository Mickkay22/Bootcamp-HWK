Presidential Approval Rating vs. Murder

This website displays my analysis on Presidential Approval Ratings vs Crime Rate over the last 60 years while practicing/showing off some of my HTML, CSS and Bootstrap skills. On this webpage you will find 5 different tabs that you can explore: Visualizations, Comparison, Data, Process, and Extra Findings.

  • Visualizations: This tab will let you choose which crime type (Total Crime, Violent, Property, Murder) you would like to see against the presendential approval rating with graphs and a description
  • Comparison: On this page you will see all the scatter graphs next to each other
  • Data: Here you'll see where the data was pulled from and the table with all combined data that was used in the analysis
  • Process: This page will show screenshots of how the anaylsis was done using Python
  • Extra Findings: Here you will see any extra analysis done with this data
